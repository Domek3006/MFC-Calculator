# A calculator application made using MFC

### The calculator functionality:

- [x] Addition
- [x] Subtraction
- [x] Multiplication
- [x] Division
- [x] Square root
- [x] Clear display
- [ ] Save result to memory
- [ ] Switch to negative number
- [ ] Session history
- [ ] Save session to file
- [ ] Add display of currently processed values
- [ ] Keyboard support
